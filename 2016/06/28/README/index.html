添加灾情，sos

POST /sosmessage/insert

参数 说明
content 内容
address 地址
image 图片，多张逗号分隔(不要前缀)
voice 语音(不要前缀)
type 类型 1=灾情 2＝sos
coordX 地图横坐标
coordY 地图纵坐标
disaster 灾害类型 1=水灾 2=火灾 3=其他
level 灾害等级 1=一级 2＝二级 3=三级 4=四级 5=五级
request
auth
sign
